Black background theme for xchat 2.4.x. Quit xchat before starting
the installation, or it will not work.

FILES:
	pevents.conf
	palette.conf
	colors.conf

UNIX: Place these files inside your ~/.xchat2/ directory.

WINDOWS: Place these files inside your $APPDATA\X-Chat 2\ directory.
APPDATA is your current user's application data folder. The location
of this directory varies on different Windows versions, but on XP it
is usually:
   C:\Documents and Settings\<username>\Application Data\X-Chat 2
Just do a search for "Application Data" and you should find it.
