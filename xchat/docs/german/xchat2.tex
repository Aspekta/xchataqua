Die Menüzeile wurde gegenüber der alten Version etwas aufgeräumt. So befinden
sich folgende Menüpunkte in der Menüzeile:
\begin{description}
  \item[Server List] Verwalten der IRC Server und Netzwerke mit denen man in
  Verbindung treten kann. Außerdem können noch Verbindungsoptionen
  eingestellt werden. \index{XChat 2 !Server List}
  
  \item[New] Über die Untermenüs, kann man neue Server- und Kanalreiter, sowie
  Fenster zum Hauptfenster des XChat hinzufügen.
  
  \item[Load Plugin or Script\dots] Neue Plugins oder Skripte können über
  diesen Menüpunkt zum XChat hinzugeladen werden. XChat kann mit Perl-, Tcl-
  und Pythonscripts erweitert werden. Abhängig ist dies jedoch von der
  Distribution und der Kompilation des Programms.
 
  \item[Detach Tab] Hier kann der aktuelle Reiter aus dem Hauptfenster ``abgetrennt''
  werden. Der Reiter erscheint dann in einem neuen Fenster.
  
  \item[Close Tab] Der aktuelle Reiter kann mit diesem Menüpunkt geschlossen werden.
  \item[Beenden] Hiermit kann XChat beendet werden.
\end{description}

\subsection{Text Box}
\begin{description}
  \item[Schriftart:] Die Schriftart die benutzt werden soll, um den Text im
  Textfenster anzuzeigen.\index{Schriftart}
  
  \item[Background Image:] Welches Bild benutzt werden soll um den Hintergrund
  des Textfensters zu bilden. \index{Hintergrundbild}
  
  \item[Time Stamp Format:] Das Format, wie eine Zeitanzeige im Textfenster
  erfolgen soll. Die Zeitanzeige erfolgt am Anfang jeder Zeile, vor dem
  Spitznamen. Standardmässig wird diese Option nicht berücksichtigt, da die
  Option \texttt{Time Stamp Text} nicht ausgewählt ist. \index{Time Stamp Format}
  
  \item[Scrollback Lines:] (Textpuffer-Grö0e) Die Nummer von Zeilen, die
  gepuffert werden sollen (0 = alle, 300 = Standardeinstellung).\index{Textpuffer}
  
  \item[Tint (Red|Green|Blue)] Die Einfärbung der Transparenz wird in RGB
  Werten geregelt.\index{Tint}
  
  \item[Transparent background] - Der Hintergrund wird
  pseudo-transparent.\index{Transparenz}

  \item[Tint transparency] - Die Transparenz wird mit einem bestimmten Farbton eingefärbt.

  \item[Time stamp text] - Hier wird vor jeder neuen Zeile die Uhrzeit im Textfenster mit ausgegeben.\index{Zeitmarkierung}

  \item[Indent nicks] - Spitznamen und Text werden mit einem Tabulator angeordnet.

  \item[Colored nicks] - Jetzt werden Spitznamen farbig
  angezeigt.\index{Spitznamen!Farben}

\end{description}

\subsection{Input Box}\index{Eingabezeile}

\begin{description}
  
  \item[Nick completion suffix:] Durch Setzen wird der eingegebene Text nach
  falschen Spitznamen durchsucht und berichtigt. Siehe auch Tab
  Spitznamen.\index{Spitznamen!Vervollständigung}

  \item[Interpret \%nnn as an ASCII value]\index{Eingabezeile!Codes}

  \item[Interpret \%C, \%B as Color, Bold etc]\label{colorcode} Durch Angabe von \verb|%C|
  gefolgt von einem Farbwert, wird der nachfolgende Text in dem angegebenen
  Farbwert interpretiert. Das Wort \verb|%B| interpretiert XChat als
  fettgedruckte Zeichen. Beispiele: \verb|%C04 testwort| wird, in der
  Standard-Einstellung,  als rotes
  ``testwort'' ausgegeben, \verb|%B testwort| wird als \textbf{testwort}
  ausgegeben.

  \item[Automatic nick completion] Durch betätigen der
  \verb|TABULATOR|-Taste wird der Spitzname vervollständigt. So reicht es aus,
  einen Spitznamen anfänglich mit mehreren Buchstaben zu schreiben und danach
  gefolgt von \verb|TAB| diesen vervollständigen zu lassen. Bei mehreren
  Übereinstimmungen, schlägt XChat die gefundenen Spitznamen im Textfenster
  vor.

  \item[Use the Text box font and colors] Die Schriftart und Farbe der Textbox
  wird auch in der Eingabezeile benutzt.

\end{description}

  \begin{itemize}

\subsection{Colors}
Hier werden Farbwerte definiert, die zum Beispiel als Farbcode in die
Benutzerzeile eingegeben werden können. Die meisten IRC-Clients können dann
diese Farben interpretieren. \textbf{Hinweis:} Je mehr Farben in einem
Gespräch Verwendung finden, desto unübersichtlicher wird die Konversation. Es
wird empfohlen, wenige Farben zu benutzen, wenn icht ganz auf Farben zu
verzichten.

\begin{description}
\item[mIRC colors:] Hier können jeder Zahl von $0-15$ ein Farbwert zugeordnet
werden. Wird z.B. als Farbwert $04$ Verwendet, entspricht dies in der
Standardeinstellung der Farbe \verb|rot|. Farben können auf zwei
unterschiedliche Arten eingefügt werden:
\begin{itemize}
  \item in der Eingabezeile per Kommando: \verb|%C<FARBWERT>|
  (\ref{colorcode})

  \item \verb|Rechter Mausklick auf den Reiter| und im Kontextmenü
  \texttt{Insert color code} den gewünschten Farbwert auswählen.

\end{itemize}

\item[Foreground:] Die Vordergrundfarbe der Textbox.

\item[Background:] Die Hintergrundfarbe der Textbox.

%\item[Mark fore:]  XXX

\item[Mark back:] Die Farbe der Markierung bei markiertem Text.

\item[New Data:] Farbe die im Reiter benutzt wird, sobald es neue Daten in dem
Reiter gibt. \textbf{Beispiel:} Jemand betritt einen Kanal.

\item[New Message:] Farbe die für den Reiter benutzt wird, um anzuzeigen, dass
eine Person eine neue Nachricht geschrieben hat.

\item[Highlight:] Die Farbe wird benutzt, um anzuzeigen, dass jemand in dem
Reiter Deinen Namen benutzt hat.

\end{description}

\subsection{Global User Info}
Die Ansicht auf die globalen Benutzer Informationen (Global User Info) bleiben
beim wechseln in die verschiedenen Ansichtsmodi erhalten. Hier können
persönliche Einstellungen vorgenommen werden. Es existieren 3 verschiedene
Felder für das Einstellen des \textbf{Nick Namen}, im folgenden auch
\textbf{Spitznamen} genannt. Im ersten Feld wird der
normal gebräuchliche Spitzname hineingeschrieben. Die zwei weiteren Felder
dienen dazu, alternative Spitznamen zu verwenden, die auf dem ersten Server schon vergeben
sind. Auf Abbildung (\ref{fig:servlistedit}) wird zum Beispiel \verb|roman|
anstatt von \verb|romanofski| verwendet, wenn \verb|romanofski| schon auf dem
zu verbindenen Server vergeben ist.


Der Benutzername (\textbf{User Name:}) und der reale Name (\textbf{Real
    Name:}) sollte ebenfalls noch spezifiziert werden. Manche Server fragen
auch diese Einstellungen ab und lassen keine Verbindung zu, falls diese Felder
nicht ausgefüllt sind.

\subsection{Betrachtermodus}

Der Betrachtermodus stellt nur die eingetragenen
Netzwerke\footnote{Nochmal zu Erinnerung: Zu jedem Netzwerk können mehrere
  oder einzelne Server gehören. Die Benutzer werden dann auf die jeweiligen
  Server im IRC Netz aufgeteilt} dar, zu denen sich eine Verbindung herstellen
lässt.

Eine Verbindung zu einem Netzwerk wird aufgebaut, wenn dieses
ausgewählt ist und der \textbf{Connect} Knopf gedrückt wird.
